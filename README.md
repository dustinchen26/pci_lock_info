## Example
Please input
PCI Number: 7
ARFCN Number: 720288
SCS: 30
Band Number(1~300): 79

Final Output:
PCI Number converted to bytes: 07000000
ARFCN Number converted to bytes: A0FD0A00
SCS converted to bytes: 0100
Band Number converted to bytes: 4E00

Final Output:
07000000 A0FD0A00 0100 4E00
